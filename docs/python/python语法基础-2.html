变量

用简单简介的英文单词来标识一个变量，由字母、数字、下划线组成，变量首字母不能为数字，系统关键字不能用在变量中。变量没有类型限制。

a = 1
b = a
a = 3
print(b)
>> 1                     # int 值类型

a = [1,2,3,4,5]
b = a
a[0] = '1'
print(b)
>> ['1', 2, 3, 4, 5]     #list 引用类型

int、str、tuple 值类型，不可变

list、set、dict 引用类型，可变

a = 'hello'
print(id(a))              #4514374688 打印a的内存地址
a = a + ' python'         #一个新的字符串
print(a)  
print(id(a))              #4513862576 打印a的内存地址

#列表
aa = [1,2,3]
print(hex(id(aa)))        # 0x10985e388
bb = aa 
print(hex(id(bb)))        # 0x10985e388

#元组 不能修改,不可改变
c = (1,2,3)
print(c)

cc = (1,2,3,['a','b','c'])
print(cc[-1][2])
>> c
cca = (1,2,3,[1,2,['a','b']])
print(cca[-1][2][1])
>> b
cca[-1][2][1] = 'xx'   #修改元组中 列表中的元素
print(cca[-1][2][1])
>> xx

#必须参数 x、y没有默认值
def need_params(x,y):
    return x+y
sum = need_params(y=1,x=2)  #关键字参数

#默认参数 y为默认参数
def default_params(x,y=1):
    return x+y
print(default_params(2))    #传一个参数即可

#可变参数 *x  一个tuple类型

    print(x)

#关键字参数 **y 接收到的是一个字典

    print(y)
